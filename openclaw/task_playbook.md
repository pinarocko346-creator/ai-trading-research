# OpenClaw 多 Agent 任务手册

## 用法

这份手册提供两类可直接复用的任务模板：

- `A股日更快讯`
- `美股深度研究`

建议把任务先发给 `总控主编`，再由它按 `openclaw/workflows.yaml` 调度。

## 模板 1：A股日更快讯

适用场景：

- 收盘后日报
- 次日盘前重点
- 午盘市场小结

建议输入：

```text
任务目标：生成今晚 A 股盘后重点与明日盘前观察
市场范围：a_share
标的列表：留空，先由 市场扫描员 自动提炼
时间窗口：最近 1 个交易日
输出格式：L2 观点短评 + L4 60秒口播稿
禁止事项：不要虚构盘后消息；不要把猜测写成确定性结论
资料来源：
- reports/openclaw_daily/latest/manifest.json
- reports/openclaw_daily/latest/daily_candidates.csv
- reports/openclaw_daily/latest/daily_report.md
```

建议执行链：

1. `总控主编` 判断是否需要聚焦在指数、板块还是个股
2. `市场扫描员` 提炼 3 到 5 个最值得跟的主题
3. `新闻情绪研究员` 提炼情绪与叙事变化
4. `总控主编` 整合成统一观点
5. `内容写手` 生成公众号短评和口播稿
6. `总控主编` 终审

推荐终稿结构：

- 开头：今天市场最重要的一个结论
- 中段：3 个重点主题或标的
- 结尾：明天最值得继续观察的指标

## 模板 2：美股深度研究

适用场景：

- 财报季个股复盘
- 行业景气度拆解
- 大主题投资逻辑梳理

建议输入：

```text
任务目标：拆解某只美股或某个行业现在是否值得持续跟踪
市场范围：us_equity
标的列表：如 NVDA、TSLA、META
时间窗口：最近 1 个季度到最近 1 年
输出格式：L3 深度长文 + L4 90秒视频脚本
禁止事项：不要引用无来源市场传闻；估值材料不足时必须单列说明
资料来源：
- 财报摘要
- 新闻整理
- 宏观或行业背景材料
- 市场异动清单
```

建议执行链：

1. `总控主编` 明确问题，例如“现在的催化是否足以支撑新一轮交易”
2. `基本面研究员` 输出业务、财报、估值和风险底稿
3. `新闻情绪研究员` 输出市场共识、分歧和误读点
4. `市场扫描员` 补充该标的在当前市场中的位置与关注度
5. `总控主编` 形成最终观点
6. `内容写手` 改写成长文和视频脚本
7. `总控主编` 终审

推荐终稿结构：

- 开头：一句话结论
- 主体 1：现在发生了什么
- 主体 2：市场为什么会交易这个逻辑
- 主体 3：最大的风险和失效条件
- 结尾：未来一到三个观察指标

## 平台改写建议

### 公众号

- 适合 `L2` 和 `L3`
- 标题用“结论 + 对象 + 原因”
- 正文尽量保留证据链

### 小红书/短视频口播

- 适合 `L1` 和 `L4`
- 开头 1 句必须给出最强钩子
- 正文控制在 3 段内

### 社媒短帖

- 适合 `L1`
- 只讲一个中心判断
- 必须保留条件和风险提示

## 最低发布门槛

任何准备发布的内容，至少要满足：

- 有明确事实来源
- 有风险点
- 有失效条件
- 没有绝对化表达
