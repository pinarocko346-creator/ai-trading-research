# 美股自动选股完整量化筛选系统

## 1. 系统定位

本系统用于美股右侧趋势交易，核心思想是：

- 先看大盘，后看个股。
- 只做已经出现右侧突破的强势股，不做左侧抄底猜底。
- 以梯子为主、CD 指标为辅；CD 负责提示，梯子负责确认。
- 所有买入、加仓、减仓、卖出都必须有明确级别与明确依据。
- 目标不是买在最低卖在最高，而是稳定吃趋势中段的“鱼身”。

本系统同时适用于：

- 正股筛选
- 月度及双月期权筛选
- 小级别短线机会观察

不适用于：

- 低流动性小票
- 陌生股票
- 纯消息驱动博弈
- 财报赌博
- 无明确梯子结构的走势

## 2. 核心定义

### 2.1 梯子定义

- 黄色梯子：长期走势
- 蓝色梯子：短期走势
- 蜡烛图：当前实时价格行为

核心理解：

- 蓝色梯子在黄色梯子之上：上涨趋势概率大
- 蓝色梯子在黄色梯子之下：下跌趋势概率大
- 蜡烛图稳定在蓝色梯子之上：短期强势
- 蜡烛图稳定在蓝色梯子之下：短期偏弱

### 2.2 CD 指标定义

- CD 抄底：提示可能开始新一轮上涨趋势
- CD 卖出：提示可能开始新一轮下跌或阶段结束

使用原则：

- CD 只负责提示，不单独构成买卖依据
- 任何有效买点，必须由蜡烛图与梯子确认
- 任何有效卖点，最终都以对应级别的梯子破位为准

### 2.3 时间级别与持仓周期

- `1/3/5/10/15分钟`：极短期，波动大，只用于精细化入场与出场
- `30分钟/1小时`：周至月级趋势
- `2/3/4小时`：更稳定的波段趋势
- `日线`：月至数月趋势
- `周线`：半年到一年以上趋势

原则：

- 趋势越大，稳定性越高
- 最佳执行级别优先从 `1小时` 起步
- 买入级别与卖出级别必须一致，不受更小级别噪音干扰

## 3. 股票池规则

### 3.1 必选过滤条件

自动选股只保留同时满足以下条件的股票：

- 只做自己熟悉、耳朵听过、长期有跟踪的股票
- 只做高流动性、高交易量、高市值股票
- 不做低市值、低流动性股票
- 研究不明白的走势直接放弃，不硬做
- 优先关注主流强势股、龙头股、机构活跃股

### 3.2 可量化股票池模板

若需要落地成程序，建议股票池先限制为以下任一集合：

- 自定义白名单
- 纳斯达克/纽交所高流动性龙头股池
- 日均成交额、期权流动性、市场关注度同时达标的股票池

建议的程序化过滤字段：

- 市值
- 日均成交额
- 日均成交量
- 期权链流动性
- 是否属于熟悉股票白名单

说明：

- “只做耳朵听过的股票”是本系统的重要前置条件，程序上建议用白名单强制实现。

## 4. 大盘优先原则

永远先看大盘，再看个股。

### 4.1 大盘风险模式

当大盘蜡烛图稳定在蓝色梯子之下，或大盘出现明确卖出并跌破对应级别蓝梯时：

- 个股抄底与买入要大幅降低积极性
- 已持有正股，破位先走
- 已持有期权，优先减仓或清仓
- 控制总仓位到保守区间
- 只保留最强个股或更大级别机会

### 4.2 大盘增强模式

当大盘与个股在同一级别一起出现抄底，并且个股完成梯子突破时：

- 个股上涨确定性更高
- 同步出现的板块股可优先排序

### 4.3 对冲思路

当大盘处于明显高风险阶段，可使用波动率相关工具对冲，但对冲是风险管理工具，不是主交易逻辑。

## 5. 仓位管理

### 5.1 总体仓位

- 正股仓位：约 `85%`
- 期权仓位：不超过总仓位 `15%`
- 末日期权：不超过总仓位 `1%`

### 5.2 单票仓位

- 单个股票总风险敞口不超过总仓位 `30%`

### 5.3 加仓原则

- 只允许分段加仓，不允许一次性满仓
- 加仓必须发生在原有逻辑继续强化时
- 每一次回踩有效支撑，才是候选加仓点
- 破位时必须承认错误，不死扛

### 5.4 期权止盈规则

- 一周内到期的期权，如果已翻倍，至少卖出一半，让剩余仓位成本转负
- 月期权或双月期权若明显盈利，也可采用“翻倍卖半”管理浮盈

## 6. 自动选股总流程

### 6.1 第一步：市场环境过滤

先判断大盘是否允许积极做多：

- 允许做多：大盘对应级别未破蓝梯，或出现抄底后重新站上蓝梯
- 保守做多：大盘中性震荡，但未形成明确破位
- 禁止激进做多：大盘卖出并跌破蓝梯

### 6.2 第二步：股票池过滤

仅保留：

- 熟悉股票白名单
- 高流动性
- 高市值
- 有期权流动性（如做期权）
- 非财报赌博窗口

### 6.3 第三步：趋势状态过滤

优先级从高到低：

- 蓝在黄上，蜡烛在蓝上
- 蓝在黄上，蜡烛回踩蓝梯
- 蓝刚上穿黄，蜡烛同步走强
- 下跌趋势后出现明确右侧突破与回踩确认

直接剔除：

- 蓝在黄下且蜡烛长期在蓝下
- 只有 CD 抄底，没有梯子确认
- 走势杂乱、梯子不清晰

### 6.4 第四步：信号归类

将候选股归入以下策略类型：

- 教程基础打法
- 蓝在黄上打法
- 1234 / 4321 多级别共振打法
- 集体行为打法
- 呼应打法
- 叶总甜点理论

### 6.5 第五步：输出执行参数

每只候选股必须同步输出：

- 触发级别
- 对应策略类型
- 买入触发条件
- 加仓条件
- 卖出参考级别
- 失效条件
- 建议使用正股还是期权
- 建议期权到期时间

## 7. 各策略的量化定义

## 7.1 教程基础打法

### 触发条件

- 某一级别出现 CD 抄底
- 蜡烛图收盘价快速钻出蓝色梯子上边缘

### 买入条件

- 收盘价突破蓝梯并站稳，可买入

### 卖出条件

- 对应级别蜡烛图收盘跌破蓝色梯子下边缘

### 适用说明

- 这是最基础、最常用的右侧确认打法
- CD 是起点，蓝梯突破是确认

## 7.2 蓝在黄上打法

### 触发条件

- 蓝色梯子已在黄色梯子之上
- 个股整体进入上涨趋势
- 回调过程中蓝梯回撞黄梯后重新拐头向上，或蜡烛短暂跌破蓝梯后迅速修复

### 买入条件

- 蓝梯重新走强，蜡烛重新站上蓝梯，可买入或买回

### 持有原则

- 该级别蓝梯持续在黄梯之上时，偏向继续持有

### 卖出条件

- 对应级别收盘跌破蓝梯下边缘

## 7.3 1234 / 4321 多级别共振打法

### 核心定义

以下条件同时满足时成立：

- `1小时、2小时、3小时、4小时` 在相近价位同步出现 CD 抄底
- 最好此前 `4小时` 级别曾在高位出现过准确卖出
- 从高位回落 `30%` 或以上更佳
- `30分钟` 级别蜡烛图收盘价快速突破蓝色梯子上边缘

### 买入条件

- `30分钟` 级别完成快速突破蓝梯后买入

### 卖出条件

- 以 `30分钟` 或 `1小时` 级别蓝梯下边缘作为卖出参考
- 站稳更大级别后，可将卖出参考升级到更大级别

### 期权匹配

- 最大有效级别为 `1小时`：可考虑双周或月期权
- 最大有效级别为 `2/3/4小时`：优先月期权
- 最大有效级别为 `日线`：优先月或双月期权

## 7.4 集体行为打法

### 触发条件

- 同一板块或同一类别股票在同一时间段、同一级别同步出现抄底
- 板块中多只核心股票同步走强
- 目标股票对应级别蜡烛图快速突破蓝梯

### 买入条件

- 目标股票完成蓝梯突破时买入

### 卖出条件

- 对应级别收盘跌破蓝梯下边缘

### 强化条件

- 如果大盘也在同一级别同步抄底，则优先级更高

## 7.5 呼应打法

### 核心逻辑

- 优先观察“上一次卖出最准确”的那个级别

### 使用方法

- 若某股票前一次准确卖出出现在 `30分钟`
- 则后续优先跟踪 `30分钟` 级别的下一次抄底与突破

### 作用

- 帮助确定当前最值得信任的观察级别

## 7.6 叶总甜点理论

叶总甜点理论分为两类高确定性结构，核心都是“阻力变支撑”或“趋势中回踩确认支撑”。

### 甜点类型 A：下跌结束后的右侧甜点

#### 前置条件

- 前一个周期的趋势明确偏弱
- 蓝色梯子稳定在黄色梯子之下
- 前期多次未能突破黄色梯子，黄色梯子已被验证为明显阻力位

#### 触发条件

- 出现 CD 抄底
- 蜡烛图强势、快速突破蓝黄梯子
- 随后回踩到黄色梯子附近

#### 买入 / 加仓条件

- 回踩黄色梯子后企稳，可作为买点或加仓点
- 若回踩时蓝色梯子已经在黄色梯子之上，确定性更高

#### 失效条件

- 回踩继续向下并踩破黄色梯子，说明支撑失效，止损离场

### 甜点类型 B：上涨趋势中的回踩甜点

#### 前置条件

- 某股票已经稳定运行在蓝色梯子之上
- 对应级别遵循“不破梯子不卖出”

#### 触发条件

- 每一次回踩蓝色梯子，都是一次验证支撑的时刻

#### 买入 / 加仓条件

- 回踩蓝梯后企稳反弹，可作为买点或加仓点

#### 失效条件

- 跌破该级别蓝色梯子下边缘，止损离场

### 甜点理论执行原则

- 只做右侧突破后的甜点，不做左侧猜底
- 只做熟悉、流动性好的股票
- 趋势越大越稳定，优先做小时级别及以上
- 买入级别与卖出级别保持一致
- 每天像逛街一样扫描股票，看到甜点时刻要果断执行

## 8. 分段建仓规则

若要将系统进一步程序化，可采用以下分段模型：

### 第一笔

- CD 抄底出现
- 蜡烛图突破蓝梯并站稳
- 建立首笔试仓

### 第二笔

- 蜡烛图突破黄梯并站稳
- 趋势从短期强势进入更高一级确认

### 第三笔

- 蓝梯钻出黄梯
- 趋势进一步强化，可做最后一次加仓

### 回踩加仓

- 蜡烛图回踩蓝梯
- 或蓝梯回撞黄梯后再次起飞
- 只要支撑有效，即可按原逻辑补仓

## 9. 卖出系统

## 9.1 通用卖出原则

- 蜡烛图不破蓝梯下边缘，不卖出
- 蜡烛图收盘跌破蓝梯下边缘，卖出
- CD 指标卖出，且蜡烛与梯子同步转弱，卖出
- CD 指标卖出，但蜡烛与蓝梯继续向上，可暂时忽略 CD

## 9.2 已有盈利时的处理

- 若收盘价明显远离梯子，且 CD 出现卖出，可分批止盈
- 期权已翻倍时优先卖半，保留趋势仓

## 9.3 买回规则

- 如果个股重新走强，尤其是蓝梯始终在黄梯之上，且蜡烛重新突破蓝梯，应快速买回
- 如果蓝梯回撞黄梯后重新起飞，也属于有效买回机会

## 10. 禁止事项

以下情况自动剔除，不进入候选池：

- 低市值、低流动性、低交易量股票
- 陌生股票
- 财报博弈
- 仅靠新闻判断涨跌
- 只因“跌很多、看起来便宜”而想买入
- 没有明确级别、没有明确买卖点的走势
- 融资重仓、杠杆失控
- 因踏空、连胜后膨胀、亏损后急于翻本而冲动交易

## 11. 自动排序优先级

当多个标的同时出现机会时，优先级从高到低建议如下：

1. 大盘与个股同级别共振抄底
2. 集体行为 + 个股突破蓝梯
3. `1234 / 4321` 多级别共振
4. 甜点类型 A：阻力变支撑
5. 甜点类型 B：上涨趋势回踩蓝梯
6. 蓝在黄上且回撞黄梯再起飞
7. 单一级别教程基础打法

加分项：

- 股票属于熟悉白名单
- 流动性极好
- 趋势级别更大
- 突破速度快
- 蓝梯与黄梯张口扩大
- 大盘环境配合

减分项：

- 大盘偏弱
- 走势磨蹭
- 仅小级别成立
- 期权流动性差
- 个股消息风险大

## 12. 期权映射规则

### 12.1 基本原则

- 期权是正股趋势的放大器，不是赌博工具
- 级别越大，期权到期时间越长
- 只在有明确趋势结构和明确卖点时使用期权

### 12.2 建议映射

- `1/3/5/10/15分钟`：仅适合极小仓位训练或末日期权观察
- `30分钟/1小时`：双周或月期权
- `2/3/4小时`：月期权
- `日线`：月或双月期权
- `周线`：双月或更长周期

### 12.3 行权价思路

- 优先选择现价附近或轻度虚值
- 月期权常用参考：现价上方约 `10%`
- 级别越大、趋势越稳，可适度拉长到期日，而不是盲目拉远行权价

## 14. 最终执行原则

### 做

- 做右侧突破
- 做熟悉股票
- 做高流动性龙头
- 做大趋势级别
- 做有明确买点、卖点、止损点的机会
- 做可以写成规则并重复执行的策略

### 不做

- 不猜顶猜底
- 不贪便宜捡漏
- 不看新闻下注
- 不赌财报
- 不临时起意
- 不重仓单票
- 不让小级别噪音破坏大级别计划

## 15. 一句话总结

这套美股自动选股系统的核心，不是“预测哪只股票会暴涨”，而是先用大盘过滤风险，再用熟悉股票池与流动性过滤标的，最后只在 CD 提示后、梯子确认的右侧突破与回踩支撑结构中出手，并用同级别梯子完成全程持有与卖出管理。
